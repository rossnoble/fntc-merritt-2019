Grading Breakdown

The grading for FITC-130 - Web Development is broken up as follows:

  HTML CSS Class Activities Capstone
Value 25% 25% 20% 30%
Summary Completion of HTML exercises Completion of CSS exercises Class discussion and activities Final website live via Github Pages

Capstone

The final project for this course is a static web site built from scratch using HTML and CSS.

  • Design
  • Content
  • Deployment via Github Pages
  • Demo

Design

The design of your website includes all the visual aspects. Each website will have its own unique personality, but there are some general concepts that apply to all. Consider the following:

  • Readability
  • "Negative" or "empty" space
  • Appropriate use of colour
  • Rhythm and balance

Deployment

Submit your website by posting the published URL on Slack. It should look something like this: https://rossnoble.github.io/fntc-merritt-2019.

Demo

Your demo is a brief (3-5 minute) and informal presentation of your website to the class. Think of it as show and tell. Here are some of things you could talk about:

  • Your inspiration for the site
  • Parts you found challenging
  • Some things you might add in the future.

HTML

This portion is based the HTML activities in the class. You goal is to demonstrate your understanding of the language. Focus on the following:

  • Parent/child relationships
  • Proper indenting
  • Matching open and close tags
  • Keep it simple!

CSS

This portion is based the CSS activities in the class. You goal is to demonstrate your understanding of the language. Focus on the following:

  • Organizing style blocks from generic (top) to specific (bottom)
  • Matching curly braces
  • Type selectors p {}
  • Class selectors .banana {}
  • Clean code!
